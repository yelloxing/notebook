String（字符串）

创建字符串可以使用单引号、双引号、三单引号和三双引号，其中三引号可以多行定义字符串，有点类似ES6中的反引号。

访问字符串中的值

方括号 []

可以使用方括号 [] 来截取字符串，字符串的截取的语法格式如下： 

    变量[头下标:尾下标]

索引值以 0 为开始值，-1 为从末尾的开始位置：

下面是一些有用的例子： 

val_str='zxl20070701'

print(val_str[:]) #zzxl20070701
print(val_str[0]) #z
print(val_str[1:]) #xl20070701
print(val_str[1:3]) #xl
print(val_str[:3]) #zxl
